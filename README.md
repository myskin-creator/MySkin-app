# MySkin App 🌿

Skincare product scanner and routine builder.

## How to deploy (step by step)

### 1. Upload to GitHub
1. Go to github.com and sign in
2. Click the **+** button top right → "New repository"
3. Name it `myskin-app`
4. Make it **Public**
5. Click "Create repository"
6. Click "uploading an existing file"
7. Drag both files (`index.html` and `manifest.json`) into the upload area
8. Click "Commit changes"

### 2. Deploy with Vercel (free)
1. Go to vercel.com
2. Sign in with GitHub
3. Click "Add New → Project"
4. Select your `myskin-app` repository
5. Click "Deploy"
6. Done! You get a free URL like `myskin-app.vercel.app`

### 3. Open on your phone
- Open the URL in Chrome (Android) or Safari (iOS)
- Tap "Add to Home Screen"
- It works like a native app!

## Features (v1)
- ✅ Onboarding with skin profile
- ✅ Barcode scanner (camera)
- ✅ Manual barcode entry
- ✅ Product lookup via Open Beauty Facts (170k+ products)
- ✅ Conflict detection (retinol + acids, vitamin C combos, etc.)
- ✅ AM/PM routine builder
- ✅ Works offline after first load
- ✅ Installable on phone (PWA)
